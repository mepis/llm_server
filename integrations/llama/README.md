# Getting Started

## Laptop Installs

1. Run this in the models/ folder: wget https://huggingface.co/bartowski/Tesslate_OmniCoder-9B-GGUF/resolve/main/Tesslate_OmniCoder-9B-IQ4_NL.gguf
2. Update the host value in the Tesslate_OmniCoder-9B-IQ4_NL.sh file to this machine's IP.
3. Run install_local_laptop.sh
